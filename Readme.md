# NumPy Tutorial

## Overview

This repository contains a beginner-friendly Jupyter Notebook that demonstrates the fundamentals of NumPy, an essential Python library for Data Science and Machine Learning.

## Topics Covered

- Creating NumPy arrays
- Array attributes
- Indexing and slicing
- Reshaping arrays
- Mathematical operations
- Broadcasting
- Aggregation functions
- Performance comparison

## Technologies Used

- Python
- NumPy
- Jupyter Notebook

## Installation

```bash
pip install numpy jupyter
```

## Running

```bash
jupyter notebook
```

Open the notebook and run all the cells.
